# Radiation_Induced_EV_Carry_Prognostic_Protein_Signatures_in_HNSCC
This R code was used for the proteomics data analysis presented in the paper “Radiation-Induced Extracellular Vesicles Carry Prognostic Protein Signatures in Head and Neck Squamous Cell Carcinoma” by Maria Bazovkina et al. (2026). Please refer to the paper for further details.
